# DASM-e | Digital Auction Specialists Markets & e-Commerce

**منصة المزادات الرقمية المتخصصة (DASM-e)** هي نظام سعودي متكامل لربط معارض السيارات والتجار والأفراد بمنظومة مزادات تفاعلية تشمل:
- 📺 بث مباشر مدعوم بالذكاء الصناعي.
- ⚙️ نظام مزايدة ذكي (مباشر، فوري، صامت).
- 💼 آلية تمويل مرنة ومحفظة داخلية ذكية.
- 🔗 تكامل مع الأنظمة الحكومية والمصرفية.

---

## 🚀 المميزات الرئيسية

- 🎥 **بث مباشر تفاعلي:** عبر YouTube Live و OBS، مع تحويل الكلام إلى نص.
- 🏷️ **أنواع مزادات متعددة:** حراج مباشر، مزاد فوري، مزاد صامت.
- 🧠 **تقييم ذكي للسيارات:** باستخدام الذكاء الصناعي لحساب السعر العادل.
- 💳 **نظام تمويل وتصفية:** يربط بين التجار والممولين بشكل مرن.
- 📲 **واجهة مستخدم عصرية:** مبنية بـ Next.js وTailwind CSS.
- 📡 **إدارة مركزية:** غرفة تحكم رقمية تدير كل البث والمزادات.

🛠️ التقنيات المستخدمة

Backend: Node.js (Express), MongoDB/PostgreSQL (قابل للتبديل)

Streaming: OBS Studio, YouTube API

AI Layer: OpenAI, Vision APIs (خطط مستقبلية)

Deployment: Vercel, GitHub CI/CD

Other Tools: WebSockets, MyFatoorah, برق، تابي
